# Welcome to Alpine Linux in the... WEB!
It's really simple. Go on the website (https://18891eaa.alpine-linux-web.pages.dev) and create a account. (Note, the only button for sign in which works is email, but if you click that it has all the other sign in options ready)        
Once a account is created login (if you have to) and wait for the thing to boot. Press enter where you see boot: press enter.       
Once you have seen localhost login: type in root. that should log you in.        

# Security   
Although there is no passoword, you can only access someone else's linux if you have their account.
